Typography

Primary Font: Source Serif Pro. Used for most headings and body copy.
Secondary Font: Source Sans Pro. Used for h3s and for emphasis in certain cases.

Summary Text. Use this to give a brief overview of the content of your page.

HEADERS

Use headers to organize your content and allow users to scan the page quickly.
Be sure to honor the hierarchy for accessibility. A user employing a screen reader to access site content will get confused if you place an h4 before an h3, for example.

h1: Page Titles

This is the largest text on the page. Only one h1 per page. Search engines give this header the highest priority weighting.

h2: Second-level section header

h2s are frequently used. You should name each section of your page with an h2. You may have several.

h2 - Section Header, For use on homepages only

On homepages only, h2s have a border around them.

h3: Third-level section header

When your h2 content is getting long, subdivide using h3s.
